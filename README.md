# FlexAnswer - An extension that integrates AI into your browser.

![FlexAnswer Logo](https://raw.githubusercontent.com/exlocus/flexanswer/refs/heads/main/readme-imgs/logo-with-text.png)

[**Русская версия / Russian Version**](README.ru.md)

**FlexAnswer** is a browser extension that integrates AI into your browser. It adds a window in the bottom-right corner for communicating with AI, allowing you to send tasks for analysis and get help solving them.

---

## 📋 Features

- **Small Window**: Adds a chat window in the bottom-right corner for sending tasks to the AI.
- **AI Model Selection**: Choose the AI model (gpt_4o or gpt_3.5_turbo) from a dropdown list.
- **IP Selection**: Allows you to host the FlexAnswer backend on a VPS/VDS server, so you don't have to run it locally every time.

## 🚀 Installation

### Installing FlexAnswer on Windows

#### What you'll need:

1.  **A computer with Windows**.
2.  **An internet connection**.
3.  An archiver, for example **7zip**.
4.  **Chrome browser** installed (or another Chromium-based browser).

### Installation Steps

#### Step 1. Download the FlexAnswer archive

1.  Open your browser and go to the [link on Github](https://github.com/exlocus/flexanswer/releases).
2.  Find the file named **Windows.zip** and click on it to download the archive.

    The archive will be downloaded to your computer's **Downloads** folder.

#### Step 2. Extract the archives

1.  Open the **Downloads** folder and find the downloaded **Windows.zip** archive.
2.  Right-click on the archive and select the **"Extract All..."** option or use any extraction program like WinRAR or 7-Zip.
3.  Choose the folder where you want to save the extracted files (e.g., your desktop).

#### Step 3. Install the application

Now let's install and run FlexAnswer.

1.  Go to the folder where you extracted the archive.
2.  Find the file **flexAnswer Installer.exe** and double-click it to run.

    The installer will open. Complete the installation and at the end, make sure the **Run FlexAnswer** checkbox is checked.

#### Step 4. Install the extension in the browser

Now let's install the FlexAnswer extension in your browser.

**For Google Chrome**:

1.  Open **Chrome** browser and type the following into the address bar:
    ```text
    chrome://extensions
    ```
2.  Turn on **Developer mode** (in the top right corner).
3.  Click the **Load unpacked** button.
4.  Find the folder where you extracted the files, then find and open the **extension** folder inside it.

**For other browsers**:

If you use a different browser, like **Firefox** or **Microsoft Edge**, find out how to enable developer mode for your browser.
